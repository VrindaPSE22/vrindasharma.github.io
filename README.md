# Vrinda Sharma Academic Website

This is a simple static website ready for GitHub Pages.

## How to publish on GitHub Pages

1. Create a new GitHub repository named `vrinda-website` or `1998vrindasharma.github.io`.
2. Upload `index.html` and the full `assets/` folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder, then save.
6. Your website will appear at the GitHub Pages URL shown there.

## To edit

- Replace the CV link in `index.html` with a direct public link to your CV PDF.
- Replace or update images in the `assets/` folder as needed.
